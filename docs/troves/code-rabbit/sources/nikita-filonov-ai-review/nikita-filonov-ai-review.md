# ai-review — Multi-Platform AI Code Review Tool

## Overview
An AI-powered code review tool supporting GitHub, GitLab, Bitbucket Cloud/Server, Azure DevOps, and **Gitea**. Built with multiple LLM providers (OpenAI, Claude, Gemini, Ollama, Bedrock, OpenRouter, Azure OpenAI).

## Key Features
- Works out of the box with GitLab, GitHub, Bitbucket Cloud/Server, Azure DevOps, and Gitea
- Multiple LLM provider support including local models via Ollama
- CI/CD integration (GitLab CI example provided)
- Docker deployment

## Platform Support
- GitHub
- GitLab
- Bitbucket Cloud
- Bitbucket Server
- Azure DevOps
- **Gitea** (explicitly listed)

## Significance
This is one of the few AI review tools that explicitly supports **Gitea** alongside the major platforms. Since Forgejo is API-compatible with Gitea, this tool likely works with Forgejo as well (though untested).
